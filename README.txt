Supply Chain Management System

📌 Overview

This project is a web-based Supply Chain Management System that uses machine learning to predict product demand and improve decision-making. It analyzes past sales data and predicts future trends using the XGBoost model.

🚀 Features

Product demand prediction (11 months data → 12th month forecast)
Data visualization (graphs & insights)
Admin dashboard (category & product management)
File upload & analysis
Chatbot for quick queries

🛠️ Tech Stack

Frontend: PHP, HTML, CSS
Backend: Python
Machine Learning: XGBoost
Database: MySQL

📊 How It Works

Upload dataset (sales data)
System processes and cleans data
Machine learning model predicts next month sales
Results shown in graphs
